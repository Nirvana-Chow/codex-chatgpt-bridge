# 公开桥隐私规则

更新时间：2026-07-09 15:37

这个 GitHub bridge 只用于给 ChatGPT 读取公开安全的系统上下文，不是 Obsidian 主库，也不是完整备份。

## 一律不上传

- 工作相关内容：真实公司、岗位细节、项目代号、供应商、同事、工作待办、工作报告和工作证据。
- 财务相关内容：账户、账单、收入、支出、投资、订阅、资产负债、现金流和财务目标。
- 朋友、人名和人物关系：真实姓名、昵称、互动记录、人物画像、关系链和私人聊天原文。
- 敏感具体事项：工作/财务/人物项目细节、私密日记、原始导出、事实表、任务表、报告索引、来源映射、执行日志和原始输入。
- 账号、密码、API Key、token、授权码、完整本机路径和未脱敏原始材料。

## 允许上传

- 不含个人细节的公开安全规则摘要。
- 用户当轮明确点名要上传的主题包，例如本次训练目标、训练计划、训记标准动作名、动作安排逻辑和给 ChatGPT 评审的训练上下文。
- 即使内容看似非敏感，也不要自动扩大上传范围；待办、AI 工作流、生活 routine、项目摘要等都必须等用户明确点名后才加入白名单。
- 本次训练主题包不得包含训记 API Key、原始训练/饮食/身体历史导出、工作、财务、人物朋友、账号凭证、本地路径或私密日记。

后续同步脚本按白名单导出。没有明确进入白名单的内容，默认不上 GitHub。
